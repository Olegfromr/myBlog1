Bootstrap

Bootstrap это свободный набор инструментов для создания веб-приложений. Состоит из CSS и HTML шаблонов, включает JavaScript расширения.

Чем Bootstrap хорош?
Это в первую очередь адаптивность. Сайт свёрстанный с помощью Bootstrap не утратит свою функциональность как на широком экране монитора, так и на экране смартфона. Не надо самому заботиться на какое устройство идёт вывод, При правильной организации всё сделается автоматически.
Во вторых Bootstrap является из самым распространённым и одним из самых простых наборов инструментов, так что при необходимости найти специалиста работающего с ним будет несложно.
С чего начать?
Ваш html документ должен обязательно включать метатеги для указания о том, что ваша страница использует HTML5 и метатег области просмотра для "отзывчивого" поведения страницы:
<!-- Кодировка веб-страницы --> <meta charset="utf-8"> <!-- Настройка viewport --> <meta name="viewport" content="width=device-width, initial-scale=1">
