Bootstrap

Bootstrap это свободный набор инструментов для создания веб-приложений. Состоит из CSS и HTML шаблонов, включает JavaScript расширения.

Чем Bootstrap хорош?
Это в первую очередь адаптивность. Сайт свёрстанный с помощью Bootstrap не утратит свою функциональность как на широком экране монитора, так и на экране смартфона. Не надо самому заботиться на какое устройство идёт вывод, При правильной организации всё сделается автоматически.
Во вторых Bootstrap является из самым распространённым и одним из самых простых наборов инструментов, так что при необходимости найти специалиста работающего с ним будет несложно.
С чего начать?
Ваш html документ должен обязательно включать метатеги для указания о том, что ваша страница использует HTML5 и метатег области просмотра для "отзывчивого" поведения страницы:
<!-- Кодировка веб-страницы --> <meta charset="utf-8"> <!-- Настройка viewport --> <meta name="viewport" content="width=device-width, initial-scale=1">
Кроме того обязательно, что бы Bootstrap работал, его надо загрузить, или указать откуда браузер сможет загрузить файлы самостоятельно:
<!-- Подключаем Bootstrap CSS --> <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.0.0-beta3/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-eOJMYsd53ii+scO/bJGFsiCZc+5NDVN2yr8+0RDqr0Ql0h+rP48ckxlpbzKgwra6" crossorigin="anonymous"> <!-- Подключаем jQuery --> <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script> <!-- Bootstrap Bundle JS с CDN включает Poper --> <script defer src="https://cdnjs.cloudflare.com/ajax/libs/twitter-bootstrap/4.6.0/js/bootstrap.bundle.min.js" integrity="sha512-wV7Yj1alIZDqZFCUQJy85VN+qvEIly93fIQAN7iqDFCPEucLCeNFz4r35FCo9s6WrpdDQPi80xbljXB8Bjtvcg==" crossorigin="anonymous"></script> <!-- Подключаем Bootstrap JS --> <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.0.0-beta3/dist/js/bootstrap.min.js" integrity="sha384-j0CNLUeiqtyaRmlzUHCPZ+Gy5fQu0dQ6eZ/xAww941Ai1SxSY+0EQqNXNE6DZiVc" crossorigin="anonymous"></script>
Bootstrap использует библиотеку jquery для облегчения написания JS скриптов, и библиотеку Popper для раскрывающегося меню, используемая здесь библиотека boodle включает в себя библиотеку Poper. Так что не забывайте включать и их. У начинающего программиста после не правильного включения этих библиотек вполне может уйти один два дня на поиск ошибки в коде. Так что будьте внимательны.
Для проверки включения можно использовать следующий код:
<nav class="navbar navbar-expand-lg navbar-light bg-light"> <button class="navbar-toggler" type="button" data-toggle="collapse" data-target="#navbarNavDropdown" aria-controls="navbarNavDropdown" aria-expanded="false" aria-label="Toggle navigation"> <span class="navbar-toggler-icon"></span> </button> <div class="collapse navbar-collapse" id="navbarNavDropdown"> <ul class="navbar-nav"> <li class="nav-item dropdown"> <a class="nav-link dropdown-toggle" href="#" id="navbarDropdownMenuLink" data-toggle="dropdown" aria-haspopup="true" aria-expanded="false"> Раскрывающееся меню </a> <div class="dropdown-menu" aria-labelledby="navbarDropdownMenuLink"> <a class="dropdown-item" href="#">Пункт 1</a> <a class="dropdown-item" href="#">Пункт 2</a> </div> </li> </ul> </div> </nav>
Если приведённый фрагмент заработал, значит bootstrap загужен нормально.

Это все требования для подключения Bootstrap к вашей веб странице. После их выполнения вы можете полностью использовать этот фреймворк.
